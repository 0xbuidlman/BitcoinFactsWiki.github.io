BTC Trader is a crossplatform opensource live charting and technical analysis desktop application. It is available for Windows, Linux and Mac OS X.

This software was announced on August 2, 2011 (1.1).

Latest version 1.4.2 includes

  • Live data feed from ?Bitcoin Charts
  • Daily data feed from Yahoo Finance

  • Rich collection of tech-analysis tools

    • 5 chart types
    • 8 annotation tools

    • 70 indicators

  • Facebook and Twitter integration

  • Automatic software updater
