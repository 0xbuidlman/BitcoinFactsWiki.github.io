clj-btc provides Clojure bindings for Bitcoin Core's JSON-RPC API.

Example use from the REPL:

user=> (require ':as btc) nil user=> (btc/getinfo) {"timeoffset" 0, "protocolversion" 70001, "blocks" 111908, "errors" "", "testnet" true, "proxy" "", "connections" 4, "version" 80500, "keypoololdest" 1380388750, "paytxfee" 0E-8M, "difficulty" 4642.44443532M, "keypoolsize" 101, "balance" 0E-8M, "walletversion" 60000}

Sources can be found on GitHub.
