Fork may refer to:

  • ?Chain fork, the occurrence of multiple blocks at the same height
  • Hardfork, a change to the protocol that makes previously invalid events valid
  • Softfork, a change to the protocol that would only invalidate previous transactions
  • Code fork, an altcoin that is a derivative of Bitcoin
