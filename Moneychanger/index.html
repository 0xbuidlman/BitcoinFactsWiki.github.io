bitcoin/ Moneychanger
  • ?Talk

A reference implementation (experimental only at present) of a currency accounting application that accesses the Open Transactions API.

Plans for the application include Bitcoin integration to support Bitcoin as one of the currencies used as the verifiable, full-reserve backing in a Degree currency trading system or payment network.

This software was announced on the Bitcoin forum on April 23rd, 2011 (1.1).
