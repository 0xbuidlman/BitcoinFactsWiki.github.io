bitcoin/ Bitcoinj

An open source Bitcoin client library built using Java and implements the Bitcoin network protocol. With version 0.7, it was renamed from BitCoinJ to bitcoinj.

The project was announced on March 7, 2011 (1.1). The project was built by ?Mike Hearn while he was a Google employee as a "20 percent time" project (1.2).
